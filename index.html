json

Json이란 Javascript 객체 문법을 따르는 문자 기반의 데이터 포맷입니다.
JSON은 문자열 형태로 존재해, 네트워크를 통해 전송할 때 아주 유용하게 사용됩니다.

개별 JSON 객체를 .json 확장자를 가진 단순 텍스트 파일에 저장할 수 있습니다. MIME 타입은 application/json 입니다.

작성 방법

이름과 값을 쌍으로 사용
자료는 (콤마),
객체는 {}
배열은 []
이름과 값엔 반드시 "" 안에 작성

그리드 선 바꾸기

그리드 선 관련 옵션들은 모두 gridLines 키에 속한 척도 옵션 아래 중첩되어 있습니다. 이 키는 축(axes)에 수직으로 그어지는 그리드 선을 커스터마이징하는 옵션을 정의합니다.

display로 차트의 그리드 선을 보이거나 숨길 수 있습니다.(초기 설정값은 true)
그리드 선의 색상은 color를 사용해 지정할 수 있습니다.
만약 그리드 선이 실선이 아닌 대시 기호이길 바란다면, 그 길이와 간격 값을 (gridLines 안에) borderDash로 구현 가능합니다.
LineWidth와 Color 키를 사용해 시작선의 굵기와 색상을 설정할 수 있습니다

scaleLabel 안에서 display 키를 써서 차트에서 척도 레이블을 보이거나 숨길 수 있습니다. (초기 설정엔 보이지 않습니다.) labelString로 텍스트를 지정할 수 있습니다. 또한, 글꼴 색상과 글꼴 집합, 크기와 스타일은 각각 fontColor, fontFamily, fontSize 을 이용해 제어할 수 있습니다. (v2.5 fontStyle은 적용되지 않습니다.)

선형 척도 설정하기

선형 척도는 수치화된 데이터를 척도로 만드는 데 사용됩니다. 대부분의 경우에 Chart.js는 자동으로 척도의 최소 및 최댓값을 감지해 나타냅니다. 사용자가 원하는 최소 및 최대값을 설정하고자 할 경우, *Axes - ticks - min&max 으로 설정할 수 있습니다.

방사형 선형 척도 설정하기

방사형 척도에서 구체적으로 표시하는 키가 많이 있습니다. 그리드 선을(원형 or 직선) 설정할 때 option - scale - lineArc를 사용하면 됩니다. 기본값은 = false, 원형은 true입니다.(true일 경우 차트로 쓰기엔 적합하지 않음)

angleLines - 차트 중심으로부터 점 레이블까지 퍼지는 선의 외관을 제어할 수 있습니다. angleLines의 색상과 너비는 color와 lineWidth 키를 사용해 제어가 가능합니다.

점 레이블 모양은 pointLabels 키를 이용해 제어 가능합니다. 이 옵션은 lineArc 값이 false로 설정될 때만 효과가 있습니다. 글꼴 색상, 글꼴 집합, 크기, 스타일은 fontColor, fontFamily, fontSize, fontStyle 키를 이용해 명시하게 됩니다.
