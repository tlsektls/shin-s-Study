jQuery-web

자바스크립트 라이브러리로 제이쿼리 셀렉터, 애니에이션 등 자바 스크립트와 동일한 작업을 수행하지만 더 쉽게 사용 가능합니다.

제이쿼리 vs 자바스크립트
  • 크로스 브라우저를 지원하기 때문에 대체 코드 불필요
  • 객체 선택 방법이 더 쉽고 정확
  • 이벤트 처리가 모든 브라우저에서 동작하며 한개의 매서드로 이루어짐
  • 애니메이션 작업을 수행하기 위한 매서드가 추가로 제공

jQuery 이론

마우스 이벤트

  • click: 클릭했을때
  • dblclick: 더블 클릭 했을 때
  • mouseover: 마우스를 위에 올렸을 때. 하위노드에 영향을 줌
  • mouseout: 마우스가 벗어났을 때. 하위 노드에 영향을 줌
  • mouseenter: 마우스를 위에 올렸을 때
  • mouseleave: 마우스가 벗어났을 때
  • mousemove: 마우스가 움직일을 때
  • hover: 마우스를 객체 위에 올렸을 때와 벗어났을 때. jQuery 전용

객체

$('객체).method(); css에서 사용된 선자의 종류 전부 객체로 사용 가능

  • jQuery('*'); 전체 선택자 탐색
  • $('#id); 아이디 선택자 탐색
  • $('.class');
  • $('type');
  • $('#id > .class, type type'); 다중 선택자(자식선택자, 하위선택자)
  • $(':hover');

탐색 traversing

탐색 매서드();

  • $().parent(); 기준 객체의 직계 부모 노드 탐색
  • $().parents(); 기준 객체의 모든 부모 노드 탐색
  • $().children(); 기준 객체의 모든 직계 자식 노드 탐색
  • $().siblings(); 기준 객체의 형제 노드 탐색
  • $().prev(); 기준 객체의 바로 이전 형제 노드 탐색
  • $().next(); 기준 객체의 바로 다음 형제 노드 탐색
  • $().prevAll(); 기준 객체의 이전 형제 노드 '모두' 탐색
  • $().nextAll(); 기준 객체의 다음 형제 노드 '모두' 탐색
  • $().find(); 기준 객체와 일치하는 하위 노드 탐색. 매개 변수 없이 사용되지 않음
  • $().filter(); 기준 객체와 일치하는 노드 탐색(걸러냄)
  • $(":first"); 첫번째 셀렉터
  • $(":last"); 마지막 셀렉터
  • $(":even"); 짝수 셀렉터
  • $(":odd") : 홀수 셀렉터
  • $(":eq(index)") : 인덱스 셀렉터(첫번째 = index값이 0부터 시작)

class 외 기타 등등 잘 분류하기

  • .classList
  • .remove(); 기준 객체부터 하위 노드까지 전부 제거
  • .empty(); 기준 객체부터 하위 노드만 제거
  • .add()
  • .addClass().클래스 선택자 적용
  • .removeClass().클래스 선택자 제거
  • .hasClass(); 기준객체에 해당 클래스가 있으면, true, 아니면 false반환
  • .eq(n)); 인덱스 번호로 객체 선택
  • .val();
  • $().html(); 기준객체의 자식 노드를 모두 리턴
  • $().text(); 기준객체의 텍스트 노드만 리턴
  • .width();
  • .height();

effect

  • .show(); 보임
  • .hide(); 숨김
  • .toggle(); show, hide 반복
  • .fadeOut(); 서서히 사라짐
  • .fadeIn(); 서서히 나타남
  • .fadeToggle(); In Out 순차 반복
  • .fadeTo(); 불투명도 조절 opacity로 서서히 사라짐
  • .slideUp(); 서서히 올라감
  • .slideDown(); 서서히 내려감
  • .slideToggle(); 반복
  • .animate(); 애니메이션 적용
  • .delay(); 대기시간 후 실행
  • .stop(); 이펙트가 끝나기 전 멈춤
